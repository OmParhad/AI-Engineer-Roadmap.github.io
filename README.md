## 📖 About The Project

AI Learning Path is an interactive web application designed to guide aspiring AI Engineers through a comprehensive learning journey. From Python fundamentals to advanced deep learning concepts, this tracker helps you stay organized and motivated with curated free resources.

### ✨ Key Features

- **📊 Progress Tracking** - Check off topics as you complete them with persistent browser storage
- **🔍 Smart Search** - Auto-search for learning materials with pre-configured queries
- **📈 Visual Analytics** - Track completion percentages for each phase and overall progress
- **🎯 Structured Learning** - 35+ topics organized into 5 progressive phases
- **📚 Curated Resources** - Free, high-quality learning materials from top platforms
- **💾 Auto-Save** - Your progress automatically saves in your browser
- **📱 Responsive Design** - Works seamlessly on desktop, tablet, and mobile

## 🚀 Getting Started

### Option 1: Use Hosted Version
Simply visit the [live demo](https://omparhad.github.io/AI-Engineer-Roadmap.github.io/) and start tracking!

### Option 2: Run Locally

1. **Clone the repository**
```bash
   git clone https://github.com/OmParhad/AI-Engineer-Roadmap.io.git
   cd AI-Engineer-Roadmap.io
```

2. **Open in browser**
```bash
   # Simply open index.html in your browser
   open index.html  # Mac
   start index.html # Windows
   xdg-open index.html # Linux
```

That's it! No build process or dependencies required.

## 💡 How to Use

1. **Browse Learning Phases** - Click on any phase to expand and view topics
2. **Check Off Topics** - Click the checkbox when you complete a topic
3. **Search Materials** - Click "Search Learning Materials" to find tutorials
4. **Track Progress** - Watch your completion percentage grow!
5. **Stay Consistent** - Your progress saves automatically in your browser

### 🎯 Pro Tips

- ✅ Build projects alongside learning to reinforce concepts
- ✅ Join AI/ML communities (Reddit, Discord, Kaggle)
- ✅ Create a GitHub portfolio showcasing your projects
- ✅ Practice coding on LeetCode for interviews
- ✅ Stay consistent - even 1 hour daily compounds over time
